# Project1

A simple Python project.

## Getting started

1. Create a virtual environment:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

2. Install packages (if needed):

```powershell
pip install -r requirements.txt
```

3. Run the sample script:

```powershell
python my_first_python.py
```
